# Bottom Sheet Example

## App

> Code shared between iOS, Android and Web

`cd packages/app`

## Expo

> Native

Expo entrypoint: `App.tsx`

`cd packages/expo`

`yarn start:expo` to start iOS and Android app with Expo

## Next.js

> Web

Next.js entrypoint: `src/pages/_app.tsx`

`cd packages/next`

`yarn dev` to start web app
